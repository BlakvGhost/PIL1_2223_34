WebP2MS est une application de gestion des emplois du temps.

Pour consulter votre emploi du temps accédez à la page principale de l'application et recherchez l'onglet ou le lien correspondant à vostre emploi du temps. Cliquez dessus pour afficher votre planning.

Pour afficher les détails d'un événement ou d'un cours dans votre emploi du temps cliquez ou appuyez sur l'événement souhaité. Les informations supplémentaires, telles que l'heure, l'emplacement ou les participants, s'afficheront alors.

Vous pouvez utiliser les options de filtrage ou de tri de l'application pour classer les événements selon certains critères. Par exemple, vous pourriez avoir des filtres pour trier par date, par type d'événement ou par catégorie. Consultez les options de filtrage ou de tri disponibles dans l'interface de l'application pour effectuer cette opération.

Pour exporter votre emploi du temps vers un format lisible ou imprimable, recherchez une fonctionnalité d'exportation dans l'application.Cela peut être un bouton "Exporter" ou une option de menu déroulant. Choisissez le format souhaité, tel que PDF ou Excel, et suivez les instructions pour générer le fichier exporté.

Pour synchroniser votre emploi du temps Vérifiez si l'application propose une fonctionnalité de synchronisation avec d'autres calendriers ou applications. Généralement, cela peut se faire en ajoutant un compte externe, tel que Google Calendar ou Outlook, dans les paramètres de l'application. Suivez les étapes indiquées pour lier les deux calendriers et synchroniser les événements.

Pour modifier Si vous n'avez pas les droits pour modifier les emplois du temps, vous devriez contacter un administrateur ou une personne ayant les droits de modification pour demander des modifications ou des ajouts d'événements. Recherchez un moyen de contacter l'équipe de support ou l'administrateur dans l'application, tel qu'un bouton "Contacter l'assistance" ou une adresse e-mail spécifiée.
